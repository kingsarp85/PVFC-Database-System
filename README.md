PVFC Database System
📖 Project Overview

This project presents the design and implementation of the PVFC database system using MySQL Workbench. It includes database creation scripts, table relationships, and SQL queries used to manage and manipulate data efficiently.

🗄️ Database Contents

The database includes:

Table creation scripts (DDL)
Data manipulation queries (DML)
Primary keys and foreign keys
Relationships between entities
Constraints and indexes
📁 Project Files
PVFC_Database.sql → Main SQL script containing full database structure and queries
PVFC_Documentation.docx → Detailed explanation of SQL scripts and queries
PVFC_ER_Diagram.pdf → Reverse-engineered ER diagram of the database system
